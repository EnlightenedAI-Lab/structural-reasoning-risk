Proof & Evidence

The exhibits below demonstrate a working, end-to-end instrument (probe → capture → persistence → structural analysis → diagnosis). Each image is included to establish that the system is real, controlled, and auditable.

01 · Controlled Probe Setup

Live evaluation console showing provider/model selection, temperature control, and evidence capture enabled prior to execution. This establishes controlled conditions before any model output is generated.

File: assets/img/01-protocol-setup.png

02 · Evidence Locker

Each run is persisted as a case record (ID + type + preview) enabling replay, inspection, and audit after the fact. Note: the “original prompt” variant is captured in a separate build and will be integrated here once the evidence set is unified.

File: assets/img/03-evidence-locker.png

03 · Cross-Model Arena Output

Multiple models responding under identical constraints, exposing structural differences (support, missing evidence, overclaiming) rather than stylistic differences.

File: assets/img/02-arena-output-stream.png

04 · Raw Forensic Artifact

Unmodified model output and metadata captured verbatim for provenance and reproducibility. This is the audit-grade “source text” the analysis is computed from.

File: assets/img/04-artifact-raw-trace.png

05 · L2 Executive Summary (Gemini)

Structural metrics computed from extracted reasoning structure (integrity, coverage, orphan rate, stability). This is the first decision-grade summary layer.

File: assets/img/05-l2-gemini-executive-summary.png

06 · L2 Executive Summary (OpenAI)

Same evaluation protocol applied to a second model under identical constraints to enable direct cross-model comparison of structural risk signals.

File: assets/img/06-l2-openai-executive-summary.png

07 · Trace X-Ray (Gemini)

Claim–premise–conclusion mapping showing where support exists, where it is missing, and which conclusions are structurally unsupported.

File: assets/img/07-l2-gemini-trace-xray.png

08 · Trace X-Ray (OpenAI)

The same X-ray view for a second model, enabling side-by-side inspection of support density and failure modes under identical prompts and constraints.

File: assets/img/08-l2-openai-trace-xray.png

09 · Risk Diagnosis + Interpreter Synthesis

Automated detection of structural vulnerability (hallucination risk, overconfidence, missing evidence) and synthesis into decision-grade interpretation across models.

File: assets/img/09-l2-risk-diagnosis.png
